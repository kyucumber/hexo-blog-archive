Spring Security - Jwt 기반 인증 처리 구현

2018-05-01 21:30:00
Backend>Java>Spring

보안 관련된 3 요소

  • 접근 주체(Principal)

보호된 대상에 접근하는 사용자

  • 인증(Authenticate)

현재 사용자가 누구인지 확인하는 과정, 보통 로그인(아이디와 비밀번호)을 이용해 인증

  • 인가(Authorize)

현재 사용자가 특정 URL, 기능 들에 접근할 수 있는 권한이 있는지 검사하는 것.

Spring Security에서는..??

Principal -> Authentication

Authenticate -> Authentication Manager

Authorize -> Security Interceptor

Authentication의 용도

  • 접근 주체 정보
  • 인증 요청 시 요청 정보를 담는다.

SecurityContext

  • Authentication을 보관한다.
  • 현재 사용자에 대한 Authentication 객체를 SecurityContext에서 받아온다.

AuthenticationManager interface

authentication 메소드를 구현하게 됨.

인증 성공 시 Authentication 객체 반환. 실패 시 AuthenticationException

AuthenticationManager는 구현된 객체가 있다.

AuthenticationManager는 가지고 있는 Provider들을 검색해서 인증을 처리한다.

이걸 이제 AuthenticationProviderBuilder를 통해서 우리가 구현한 AuthenticationProvider를 등록해주면 된다.
